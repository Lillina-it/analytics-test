# Решение тестового задания

Автор: Алина Кучина

Время выполнения: около 2 часов.

## Описание

Для решения задачи:

- сформировала сессии пользователей по всем событиям (новая сессия начинается, если между соседними событиями прошло более 5 минут);
- оставила только события выбора шаблона (`template_selected`);
- с помощью оконной функции `LAG()` сравнила каждый выбранный шаблон с предыдущим внутри одной сессии;
- подсчитала количество повторных выборов каждого шаблона;
- вывела 5 шаблонов с наибольшим количеством повторений.

Файл с SQL-запросом: `solution.sql`.
