## Описание

В рамках задания был написан SQL-запрос, который:

- формирует сессии пользователей (новая сессия начинается, если между соседними событиями прошло более 5 минут);
- находит повторные выборы одного и того же шаблона в рамках одной сессии;
- подсчитывает количество таких повторений;
- выводит 5 наиболее часто повторяющихся шаблонов.
