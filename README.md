## Описание решения

В рамках задания необходимо было определить 5 шаблонов, которые пользователи выбирали два и более раза подряд в течение одной сессии.

### Ход решения

1. Для каждого пользователя были отсортированы все события по времени.
2. С помощью оконной функции `LAG()` было определено время предыдущего события.
3. Если разница между событиями превышала 5 минут, начиналась новая сессия.
4. После формирования сессий были оставлены только события `template_selected`.
5. Внутри каждой сессии текущий шаблон сравнивался с предыдущим.
6. Если шаблон совпадал с предыдущим, он считался повторным выбором.
7. Для каждого шаблона было подсчитано количество таких повторений.
8. Результат отсортирован по убыванию, выведены первые пять шаблонов.
